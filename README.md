# Simple HR Software

A lightweight HR system built with FastAPI + SQLModel + Jinja2.

Features:
- Login system (admin / manager / employee)
- Employee directory
- Timesheets (self-entry + manager approval)
- Basic hiring / onboarding placeholders
- Deploy-ready for AWS Lightsail (nginx + systemd)
